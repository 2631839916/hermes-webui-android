📐 布局结构

界面组成

  • 顶部标题栏 - 菜单、Logo、标题、操作按钮
  • 左侧导航栏 (Rail) - 11个功能图标按钮
  • 主内容区域 - 根据导航切换面板
  • 侧边栏 (Drawer) - 会话列表
  • 底部输入区 - 附件、输入框、发送按钮

导航面板

  • 💬 Chat - 聊天界面
  • 📋 Tasks - 任务管理
  • 📊 Kanban - 看板视图
  • 🎯 Skills - 技能管理
  • 🧠 Memory - 记忆管理
  • 📁 Spaces - 工作空间
  • 👤 Profiles - 配置文件
  • ✅ Todos - 待办事项
  • 📈 Insights - 数据洞察
  • 📄 Logs - 日志查看
  • ⚙️ Settings - 设置
